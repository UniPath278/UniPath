**UniPath** is a unified multimodal framework that integrates hierarchical whole-slide image (WSI) encoding with multimodal instruction tuning, supporting both discriminative tasks (diagnosis, biomarker prediction, treatment response, survival prognosis) and generative tasks (pathology report generation, visual question answering). It is trained on 94,715 WSI–report pairs and 720,595 VQA pairs spanning 13 anatomical sites and 32 cancer types, with external validation on 33,000+ WSIs from 51 hospitals.

## Getting Started

### 1. Download Model Weights

Request access and download from [Hugging Face]().

```python
from huggingface_hub import login
from transformers import AutoModel

login()  # https://huggingface.co/settings/tokens
model = AutoModel.from_pretrained('xxx/UniPath', trust_remote_code=True)
```

### 2. Slide Embedding Extraction

UniPath extracts multi-scale slide representations from patch features and coordinates.

```python
import torch
import h5py

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = model.to(device)

# Load patch features from h5 file
file = h5py.File('path/to/features.h5', 'r')
features = torch.from_numpy(file['features'][:])
coords = torch.from_numpy(file['coords'][:])

with torch.autocast('cuda', torch.float16), torch.inference_mode():
    slide_embedding = model.encode_slide(features.to(device), coords.to(device))
```

### 3. Report Generation

```python
report = model.generate_report(slide_embedding)
print(report)
```

### 4. Visual Question Answering

```python
answer = model.answer_question(slide_embedding, question="What is the histological subtype?")
print(answer)
```

### 5. Zero-shot Classification

```python
class_names = ["adenocarcinoma", "squamous cell carcinoma", "small cell carcinoma"]
probs = model.zero_shot_classify(slide_embedding, class_names)
```
